# Gurgaon Real Estate Market Analysis

## Project Overview
Analysed 500+ residential property listings across Gurgaon using Python to answer 10 business questions on pricing, locality trends, RERA impact, and builder reputation.

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)

## Business Questions Answered
1. Which is the costliest flat?
2. Which locality has the highest average price?
3. Which locality has the highest rate per sqft?
4. Do ready-to-move properties cost more?
5. Do RERA-approved properties command a premium?
6. How does area impact price?
7. Which BHK configuration is most expensive?
8. Which property type is costliest?
9. Do certain builders price higher?
10. Are larger homes more expensive per sqft?

## Key Insights
- Premium localities consistently show higher prices and rate per sqft
- RERA-approved properties command a price premium
- Builder reputation significantly impacts pricing
- Larger homes are not always more expensive per sqft
